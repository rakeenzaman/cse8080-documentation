Getting started

To run this project you will need Node.js and Clang.

macOS/Linux/WSL Instructions:

Install Node.js

Install Homebrew.

Run the following commands:

brew install llvm@14

brew install cmake

Then clone this repository

cd to the main directory of this project (cse8080)

Run the following commands:

mkdir build

cd build

cmake ..

make

cd ..

npm install

npm start

The app should start on localhost:3000 in your default browser, you're all good to go!
